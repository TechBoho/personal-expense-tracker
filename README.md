# Personal Expense Tracker

A modern full-stack expense tracking application built with the MERN stack.

## Live Demo

Frontend (Vercel):
https://personal-expense-tracker-lemon.vercel.app

Backend (Render):
https://personal-expense-tracker-2-o8c3.onrender.com

---

## Features

- User authentication (Register/Login)
- Protected dashboard routes
- Add expenses
- Edit expenses
- Delete expenses
- Expense search
- Category filtering
- CSV export
- Interactive charts
- Monthly budget tracking
- Dark/Light mode toggle
- Toast notifications
- Responsive design

## Installation

### Clone Repository

```bash
git clone https://github.com/TechBoho/personal-expense-tracker.git
```

### Frontend Setup

```bash
cd client
npm install
npm run dev
```

### Backend Setup

```bash
cd server
npm install
npm start
```

---

## Environment Variables

Create a `.env` file inside the server folder:

```env
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
PORT=1000
```
